What can I do ?

  • Upload a Song
  • Cut your favourite portion from it
  • Share it

To be installed

  • mp3splt # For splitting mp3s
  • id3lib # For reading metadatas from mp3s

External services

  • Thinking-sphinx ( Search server )
  rake ts:in

  rake ts:start ( For indexing / searching )

  rake ts:stop
  • memcached start ( For realtime information exchange )
  memcached # starts the memcache daemon
  • Google URL shortening API
  • Facebook posting and sharing - fbgraph gem
  • Twitter sharing - twitter gem
  • fpcalc for finding fingerprints of mp3
  • acoustid for retreiving information from music database
